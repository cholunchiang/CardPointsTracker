## 🚀 Quick Setup (3 Steps)

### **Step 1: Open in iPhone Safari**
1. Open Safari on your iPhone
2. Go to: `https://cholunchiang.github.io/CardPointsTracker/dist/CardPointsTracker.html`
3. Wait for app to load (may take 30 seconds on first load)

### **Step 2: Import Your Cards**
1. First, transfer your backup file to iPhone:
   - **Via AirDrop:** Send `CardPointsTracker_MyCards_Backup.json` from Mac to iPhone
   - **OR via iCloud:** Copy file to iCloud Drive, access from Files app
   
2. In the app:
   - Tap "📊 Summary" at bottom
   - Scroll down to "☁️ Backup Status"
   - Tap "⬇️ Import Backup" button
   - Select `CardPointsTracker_MyCards_Backup.json`
   - Confirm import

3. Your 3 cards will appear! ✅

### **Step 3: Add to Home Screen**
1. While in Safari, tap the **Share** button (bottom middle)
2. Scroll down and tap **"Add to Home Screen"**
3. Name it: `CardPoints` (or whatever you prefer)
4. Tap **"Add"**
5. Done! You now have a home screen app! 🎉

## 🆘 Troubleshooting

**App doesn't load?**
- Wait 2 minutes (GitHub Pages deployment delay)
- Force refresh: Pull down in Safari
- Check URL is correct

**Can't import backup?**
- Make sure file is on iPhone (Files app)
- Use Safari (not Chrome)
- File must be .json format

**Data disappeared?**
- Did you clear Safari data?
- Check if you're in Private Mode
- Import backup to restore
